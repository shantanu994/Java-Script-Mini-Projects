# JavaScript Mini Projects

A collection of **5 JavaScript mini projects** built for practice and fun.  
Covers DOM manipulation, localStorage, and basic API integration.

---

## Project List

### 1. Random Quote Generator 
- Shows a random motivational quote each time you click a button.
- Can use a local array of quotes or a free quotes API.
- **Folder:** [`random-quote-generator`](./random-quote-generator)

---

### 2️. Background Color Changer 
- Click to change the background color to a random value.
- Displays the HEX code of the color.
- Smooth transition effect.
- **Folder:** [`background-color-changer`](./background-color-changer)

---

### 3️. Expense Tracker 
- Add income & expenses.
- Calculates balance, total income, and total expenses.
- Saves all data in **localStorage** so it persists after reload.
- **Folder:** [`expense-tracker`](./expense-tracker)

---

### 4️. Memory Card Game 
- Flip two cards to find matching pairs.
- Tracks number of moves and time taken.
- Displays a win message when all pairs are found.
- **Folder:** [`memory-card-game`](./memory-card-game)

---

### 5️. Music Player 
- Play, pause, next, and previous controls.
- Displays song progress and time.
- Supports playlists.
- **Folder:** [`music-player`](./music-player)
